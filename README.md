# MoodFlow

MoodFlow est une application web de suivi émotionnel destinée à évoluer vers une PWA de bien-être personnel.

## Structure

- `assets/` : images, icônes, logos et futures illustrations.
- `css/` : architecture CSS modulaire.
- `js/` : scripts JavaScript de l'application.
- `pages/` : pages internes de l'application.
- `index.html` : page d'entrée compatible GitHub Pages.
